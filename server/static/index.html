PSGI/Plack REST API Template

Get Started

Add REST resources in server/default.conf:

RestMap:
  v1:
    /: Rest::Root

Restart server:

cd server; twiggy -p 5055 server.psgi

Go to the server api:

http://localhost:5055/demoProject/api/v1
